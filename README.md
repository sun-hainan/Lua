## 🎯 教程特色

- **问题驱动**：每章5-8个核心问题，围绕问题展开讲解
- **深入原理**：从表象到本质，剖析语言设计思想
- **对比学习**：与其他语言对比，加深理解
- **完整示例**：每个概念都有可运行的代码示例

## 🚀 快速开始

### 运行单个文件
```bash
lua 01_hello_world.lua
```

### 使用LuaJIT（推荐，性能更优）
```bash
luajit 01_hello_world.lua
```

### 在交互式解释器中测试
```bash
lua -i 01_hello_world.lua
```

## 🔧 开发环境

- **Lua 5.4** - 最新稳定版
- **LuaJIT 2.1** - 高性能实现（推荐）

### 常用工具
- **LuaRocks** - 包管理器
- **luacheck** - 静态分析
- **busted** - 测试框架

### 安装依赖
```bash
luarocks install luasocket
luarocks install luafilesystem
luarocks install busted
luarocks install luacheck
```

## 📖 学习路径

1. **入门**：先看1-4章，理解基础
2. **核心能力**：5-8章，掌握错误处理、IO、数据库、网络
3. **数据结构**：9-10章，深入table和迭代器
4. **进阶**：11-17章，函数式、元表、模块、协程、内存管理
5. **工程**：18-20章，并发、测试、工具链
6. **扩展**：21章，C集成与FFI
