## 1. 算术运算符

### 生活类比

> Lua 的 `/` 就像精确量杯——倒进去 5，倒出来 2.5，公平但不一定是你想要的结果。如果你要"整除"，用 `//`。

### 概念讲解

Lua 算术运算符：

| 运算符 | 说明 | 注意 |
|--------|------|------|
| `+ - * /` | 标准算术 | `/` 是浮点除法，`5/2 = 2.5` |
| `//` | 整数除法（Lua 5.3+） | `5 // 2 = 2` |
| `%` | 取模 | `10 % 3 = 1` |
| `^` | 指数（右结合） | `2 ^ 3 = 8`，`2 ^ 2 ^ 3 = 2 ^ 8 = 256` |
| `#` | 长度运算符 | `#arr`、`#str` |

### 代码示例（逐行注释）

```lua
local a, b = 10, 3

print("10 + 3 =", a + b)          -- 13
print("10 - 3 =", a - b)          -- 7
print("10 * 3 =", a * b)          -- 30
print("10 / 3 =", a / b)          -- 3.3333...（浮点除法！）
print("10 // 3 =", a // b)        -- 3（整数除法，Lua 5.3+）
print("10 % 3 =", a % b)          -- 1
print("10 ^ 2 =", a ^ 2)          -- 100

-- 长度运算符
local arr = {1, 2, 3, 4, 5}
print("#arr =", #arr)             -- 5

local str = "hello"
print("#str =", #str)            -- 5
```

### ASCII 图示

```
算术运算符对比：

Lua:         5 / 2  = 2.5    （浮点除法）
Python:      5 / 2  = 2.5    （浮点除法）
             5 // 2 = 2      （整数除法）
Rust:        5 / 2  = 2      （整数除法）
             5.0 / 2 = 2.5   （浮点需显式）
Go:          5 / 2   = 2      （整数除法）
             float64(5) / 2 = 2.5
C++:         5 / 2   = 2      （整数除法）
             5.0 / 2 = 2.5

^ 指数（右结合）：
2 ^ 2 ^ 3  =  2 ^ (2 ^ 3)  =  2 ^ 8  =  256
```

### 常见错误

```lua
-- ❌ 误以为 / 是整数除法
print(10 / 5)  -- 2.0（是浮点数，不是整数）

-- ✅ 整数除法用 //
print(10 // 5) -- 2（Lua 5.3+）

-- ❌ 混淆 ^ 和 **
-- Lua: 2 ^ 3 = 8
-- Python: 2 ** 3 = 8, 2 ^ 3 = 1 (XOR)
```

## 2. 赋值运算符与多目标赋值

### 生活类比

> 赋值就像"把右边的东西放进左边的盒子"——但 Lua 先把右边全部准备好，再一次性放进左边。所以 `a, b = b, a` 能直接交换，不需要临时变量。

### 概念讲解

- Lua 赋值是**语句**，不是表达式（不返回值）
- **多目标赋值**：先计算右侧，再赋值给左侧
- 支持交换：`a, b = b, a`
- **长度赋值**：`t[#t + 1] = value`

### 代码示例（逐行注释）

```lua
-- 多目标赋值
local x, y = 1, 2
print("before swap:", x, y)  -- 1  2
x, y = y, x                   -- 直接交换，无需 temp
print("after swap:", x, y)    -- 2  1

-- 返回多个值（函数可返回多个值）
local function minmax(arr)
    return math.min(table.unpack(arr)), math.max(table.unpack(arr))
end

local m, n = minmax({3, 1, 4, 1, 5, 9, 2, 6})
print("min, max =", m, n)     -- 1  9

-- 长度赋值（尾部追加）
local t = {1, 2, 3}
print("before:", #t)          -- 3
t[#t + 1] = 4                 -- 追加
print("after append:", #t)    -- 4
```

### ASCII 图示

```
赋值过程：

a, b = b, a

Step 1: 计算右侧 (b, a) → (2, 1)
Step 2: 赋值给左侧 a, b

等价于（其他语言需要临时变量）：
temp = a
a = b
b = temp
```

### 常见错误

```lua
-- ❌ 赋值不返回值
local function test()
    local x = (a = 10)  -- ❌ a = 10 是语句，不返回值
end

-- ❌ 变量数量不匹配
local a, b = 1, 2, 3  -- 3 被丢弃
local a, b = 1         -- b = nil
```

## 3. 关系运算符

### 生活类比

> `~=` 在 Lua 里是"不等于"，不是 `!=`。这来自数学的"不等于"符号 `≠`，Lua 用 `~` 和 `=` 组合表示。

### 概念讲解

Lua 关系运算符：

| 运算符 | 说明 | 注意 |
|--------|------|------|
| `==` | 相等 | 比较值（和类型） |
| `~=` | 不等于 | **不是 `!=`！** |
| `< > <= >=` | 大小比较 | 字符串按字典序 |
| 两侧类型不同 | 返回 false | 不会报错 |

### 代码示例（逐行注释）

```lua
-- 基本比较
print("5 == 5:", 5 == 5)          -- true
print("5 ~= 3:", 5 ~= 3)          -- true（不等于用 ~=）
print("5 < 10:", 5 < 10)          -- true
print("5 > 3:", 5 > 3)            -- true
print("5 <= 5:", 5 <= 5)          -- true
print("5 >= 5:", 5 >= 5)          -- true

-- 字符串比较
print("'apple' < 'banana':", "apple" < "banana")  -- true（字典序）
print("'hello' == 'hello':", "hello" == "hello")  -- true

-- 不同类型比较（返回 false，不会报错）
print("1 == '1':", 1 == "1")      -- false（类型不同）
print("1 < 'hello':", 1 < "hello")  -- false

-- nil 比较
print("nil == nil:", nil == nil)  -- true
print("nil == false:", nil == false)  -- false（类型不同）
```

### 常见错误

```lua
-- ❌ 误用 != 而不是 ~=
-- if x != 5 then        -- ❌ 语法错误
if x ~= 5 then           -- ✅ 正确

-- ❌ 混淆 == 和 =
-- if x = 5 then         -- ❌ 赋值，不是比较！
if x == 5 then          -- ✅ 正确
```

## 4. 逻辑运算符与短路求值

### 生活类比

> `and` 就像"两者都要"——如果第一个是假的，直接说"假的"，不用看第二个。`or` 就像"至少一个"——如果第一个是真的，直接说"真的"，不用看第二个。

### 概念讲解

Lua 逻辑运算符：**关键字**，不是符号！

| 运算符 | 说明 | 注意 |
|--------|------|------|
| `and` | 逻辑与 | 返回**实际值**，不是布尔 |
| `or` | 逻辑或 | 返回**实际值**，不是布尔 |
| `not` | 逻辑非 | 总是返回布尔 |

**重要特性**：
- `and` 和 `or` 返回**实际值**，不是布尔
- **短路求值**：条件满足时不再计算右侧
- nil 和 false 是"假"，其他都是"真"（包括 `0` 和空字符串 `""`）

语义：
- `a and b`：若 a 为假返回 a，否则返回 b
- `a or b`：若 a 为真返回 a，否则返回 b

### 代码示例（逐行注释）

```lua
-- and 和 or 返回实际值（不是布尔！）
print("true and 5:", true and 5)       -- 5
print("false and 5:", false and 5)     -- false
print("nil and 5:", nil and 5)          -- nil
print("0 and 5:", 0 and 5)              -- 5（0 是真！）

print("true or 5:", true or 5)          -- true
print("false or 5:", false or 5)        -- 5
print("nil or 5:", nil or 5)            -- 5
print("0 or 5:", 0 or 5)               -- 0（0 是真，返回 0）

-- not 总是返回布尔
print("not nil:", not nil)              -- true
print("not false:", not false)         -- true
print("not 0:", not 0)                 -- false（0 是真）
print("not '':", not '')               -- false（空字符串是真）
print("not 'hello':", not 'hello')     -- false

-- 短路求值
local function side_effect()
    print("side effect!")  -- 有副作用
    return false
end

print("testing and (false first):")
if false and side_effect() then end  -- 不执行 side_effect

print("testing and (true first):")
if true and side_effect() then end   -- 执行 side_effect
```

### ASCII 图示

```
and 和 or 的返回值：

a and b:
  a 为假 → 返回 a（短路，不看 b）
  a 为真 → 返回 b

a or b:
  a 为真 → 返回 a（短路，不看 b）
  a 为假 → 返回 b

真值表（Falsy：nil, false）：
┌─────────┬─────────┬─────────┬─────────┐
│    a    │    b    │ a and b │ a or b  │
├─────────┼─────────┼─────────┼─────────┤
│  false  │  false  │  false  │  false  │
│  false  │  true   │  false  │  true   │
│  true   │  false  │  false  │  true   │
│  true   │  true   │  true   │  true   │
└─────────┴─────────┴─────────┴─────────┘
注意：Lua 中 0 和 "" 是真！
```

### 常见错误

```lua
-- ❌ 误以为 Lua 的 and/or 返回布尔
local x = true and 5
print(x == true)   -- false！（x = 5）

-- ✅ 需要布尔时用 not
local x = true and 5
print(x and true)  -- true（x=5 是真）

-- ❌ 忘记 0 是真
if not 0 then     -- ❌ 永远不会执行（0 是真）
    print("impossible")
end
```

## 5. 三元运算符替代方案

### 生活类比

> Lua 没有 `?:` 三元运算符，就像厨房里没有电饭煲——你得用普通锅加水量米，步骤多一点，但照样能煮出饭。

### 概念讲解

**Lua 没有 `?:` 三目运算符**。

替代方案：
- `condition and a or b`
- `if` 语句

**陷阱**：当 a 可能是假值时，`and/or` 组合会出问题。

### 代码示例（逐行注释）

```lua
-- and/or 实现三元（注意陷阱）
local condition = true
local result = condition and "yes" or "no"
print("condition=true:", result)  -- yes

condition = false
result = condition and "yes" or "no"
print("condition=false:", result)  -- no

-- 陷阱：当可能值是假时
local value = 0
result = value and "truthy" or "falsy"
print("value=0:", result)  -- 错误！应该是 "truthy"

-- ✅ 正确做法：用 if
if value ~= 0 then
    result = "truthy"
else
    result = "falsy or zero"
end
print("correct handling:", result)
```

### 常见错误

```lua
-- ❌ 陷阱：a 是假值时
local a = false
local b = "fallback"
local result = true and a or b  -- "fallback"（错误！应该返回 false）

-- ✅ 安全做法
local function ternary(cond, a, b)
    if cond then return a else return b end
end
```

## 6. 位运算符（Lua 5.3+）

### 生活类比

> 位运算就像"拆开糖果包装纸数里面有几颗"——把数字拆成二进制位，然后一位一位地操作。适合做标志位、权限系统。

### 概念讲解

Lua 5.3+ 位运算符（需要整数，不是浮点）：

| 运算符 | 说明 |
|--------|------|
| `&` | 按位与 |
| `|` | 按位或 |
| `~` | 按位异或（注意：不是取反） |
| `<<` | 左移 |
| `>>` | 右移 |

### 代码示例（逐行注释）

```lua
local a, b = 0b1100, 0b1010  -- 12, 10

-- 按位操作
print(string.format("0x%x & 0x%x = 0x%x", a, b, a & b))  -- 0x8
print(string.format("0x%x | 0x%x = 0x%x", a, b, a | b))  -- 0xE
print(string.format("0x%x ~ 0x%x = 0x%x", a, b, a ~ b))  -- 0x6

-- 移位
local shifted = 1 << 4  -- 16
print("1 << 4 =", shifted)  -- 16
print("16 >> 2 =", 16 >> 2)  -- 4

-- 常见技巧：判断某一位
local flags = 0b10101
local bit2_set = (flags & (1 << 2)) ~= 0
print("bit 2 of 0b10101 is set:", bit2_set)  -- true
```

### 常见错误

```lua
-- ❌ 浮点数不能做位运算
-- print(5.5 & 3)  -- ❌ 错误

-- ✅ 先转整数
print(math.floor(5.5) & 3)  -- 1

-- ❌ ~ 不是取反（取反要用异或）
-- ~5 在 C 中是 -6，但在 Lua 5.3 中 ~ 是 XOR
print(~0)  -- -1（按位全1）
```

## 7. 运算符优先级

### 生活类比

> 优先级就像"先乘除后加减"——Lua 的指数 `^` 最高，所以 `2^2^3` 是 `2^(2^3)` 而不是 `(2^2)^3`。

### 概念讲解

Lua 优先级（从高到低）：
1. `^`（指数，右结合）
2. `not # -`（一元）
3. `* / // %`
4. `+ -`（二元）
5. `..`
6. `< > <= >= ~= ==`
7. `and`
8. `or`

**建议：复杂表达式用括号**

### 代码示例（逐行注释）

```lua
-- ^ 是右结合的
print("2 ^ 2 ^ 3 =", 2 ^ 2 ^ 3)       -- 256 = 2 ^ 8
print("(2 ^ 2) ^ 3 =", (2 ^ 2) ^ 3)   -- 64 = 4 ^ 3

-- 建议用括号
local result = (2 + 3) * 4
print("(2+3)*4 =", result)            -- 20

-- 字符串拼接优先级低于算术
local s = "hello" .. ("world" or "")
print("'hello' .. 'world' =", s)
```

### ASCII 图示

```
表达式 2 ^ 2 ^ 3 的解析：

Lua（^ 右结合）：
2 ^ (2 ^ 3) = 2 ^ 8 = 256

其他语言如 Python（** 右结合）：
2 ** 2 ** 3 = 2 ** 8 = 256
```

## 五语言对比

| 运算符 | Lua | Python | Rust | Go | C++ |
|--------|-----|--------|------|-----|-----|
| 相等 | `==` | `==` | `==` | `==` | `==` |
| 不等于 | `~=` | `!=` | `!=` | `!=` | `!=` |
| 浮点除法 | `/` | `/` | `/`（int）/ `as f64` | `/`（float） | `/`（double） |
| 整数除法 | `//` | `//` | `/`（int） | `/` | `/`（int） |
| 指数 | `^` | `**` | `.pow()` | `math.Pow` | `pow()` |
| 三元 | 无（and/or） | `x if c else y` | `if c { a } else { b }` | 无（if） | `c ? a : b` |
| and/or | 关键字，返回值 | 关键字，返回值 | `&& \|\|` | `&& \|\|` | `&& \|\|` |
| falsy | `nil, false` | `None, False, 0, ""` | `false` | `false` | `false, nullptr` |

## 本章小结

| 运算符 | Lua 行为 |
|--------|---------|
| `+ - * /` | 标准算术，`/` 是浮点除法 |
| `//` | 整数除法（Lua 5.3+） |
| `%` | 取模 |
| `^` | 指数，右结合 |
| `#` | 长度（数组/字符串） |
| `== ~=` | 相等/不等（`~=` 不是 `!=`） |
| `< > <= >=` | 大小比较，字符串按字典序 |
| `and or not` | 逻辑运算符，**返回实际值而非布尔** |
| `& \| ~ << >>` | 位运算（Lua 5.3+） |
| 三元 | 无，用 `and/or` 组合或 `if` |
