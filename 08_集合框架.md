## 1. Table 作为 List（动态数组）

### 生活类比

> Lua 的 List 就像"自动扩展的抽屉柜"——你往里塞东西，它自动长出更多抽屉。索引从 1 开始（不是 0），每个抽屉编号清楚。

### 概念讲解

Lua 的 table 就是动态数组：
- `#t` 获取长度
- `table.insert` / `table.remove` 操作
- 索引从 **1** 开始

### 代码示例（逐行注释）

```lua
-- 创建列表
local list = {10, 20, 30, 40, 50}
print("initial:", table.concat(list, ", "))  -- 10, 20, 30, 40, 50

-- 添加元素
table.insert(list, 60)
print("after insert 60:", table.concat(list, ", "))

-- 在指定位置插入
table.insert(list, 3, 25)  -- 在索引 3 插入
print("after insert 25 at 3:", table.concat(list, ", "))

-- 访问
print("list[1]:", list[1])              -- 第一个
print("list[#list]:", list[#list])      -- 最后一个

-- 删除
table.remove(list, 3)  -- 删除索引 3
print("after remove 3:", table.concat(list, ", "))

-- 删除最后一个
table.remove(list)
print("after remove last:", table.concat(list, ", "))

-- 遍历
print("for loop:")
for i = 1, #list do
    print(string.format("  [%d] = %d", i, list[i]))
end
```

## 2. Table 作为 Map（字典）

### 生活类比

> Map 就像"书架上的标签抽屉"——每个抽屉有名字（键），里面放东西（值）。你可以用名字快速找到抽屉，不用数第几个。

### 概念讲解

Map 特性：
- 键值对存储
- 任意类型作为键（除 nil）
- 无序（可以用 pairs 或排序后遍历）

### 代码示例（逐行注释）

```lua
-- 创建字典
local dict = {
    name = "Alice",
    age = 30,
    ["city"] = "Beijing",  -- 等价于 city = "Beijing"
}

-- 访问（两种方式）
print("dict.name:", dict.name)            -- Alice
print("dict['age']:", dict["age"])        -- 30
print("dict.city:", dict.city)            -- Beijing

-- 赋值
dict.job = "engineer"
print("after assignment:", dict.job)

-- 删除（设为 nil）
dict.job = nil
print("after delete:", dict.job)

-- 遍历（pairs，无序）
print("pairs:")
for k, v in pairs(dict) do
    print(string.format("  %s = %s", tostring(k), tostring(v)))
end

-- 判断键是否存在
print("name exists:", dict.name ~= nil)
print("job exists:", dict.job ~= nil)

-- 安全的访问
local function safe_get(t, key)
    if t[key] ~= nil then
        return t[key]
    end
    return nil
end
```

## 3. Set（集合）

### 生活类比

> Lua 没有 Set，就像没有"无重复的篮子"——你得自己用抽屉（key = true）模拟。值变成抽屉的名字，叫"apple"的抽屉里放个"true"表示"有苹果"。

### 概念讲解

Set 特性：
- 唯一元素
- 快速查找
- 无序

Lua 实现：用 table 作为 set（值作为键，值为 true）

### 代码示例（逐行注释）

```lua
-- 创建集合
local function Set(...)
    local set = {}
    for _, v in ipairs({...}) do
        set[v] = true  -- 值作为键，true 作为值
    end
    return set
end

local fruits = Set("apple", "banana", "cherry", "apple")  -- 重复被忽略
print("has apple:", fruits["apple"])  -- true
print("has orange:", fruits["orange"])  -- nil

-- 集合运算：并集
local function union(a, b)
    local result = {}
    for k in pairs(a) do result[k] = true end
    for k in pairs(b) do result[k] = true end
    return result
end

-- 集合运算：交集
local function intersection(a, b)
    local result = {}
    for k in pairs(a) do
        if b[k] then result[k] = true end
    end
    return result
end

local a = Set(1, 2, 3, 4)
local b = Set(3, 4, 5, 6)
local u = union(a, b)
local inter = intersection(a, b)
print("union has 3:", u[3])        -- true
print("intersection has 3:", inter[3])  -- true
```

## 4. 队列（Queue）与栈（Stack）

### 生活类比

> 队列就像"排队买票"——先来的先买（FIFO）。栈就像"叠盘子"——最后放的先拿（LIFO）。

### 概念讲解

- **队列**（FIFO）：`enqueue` 尾插，`dequeue` 头删
- **栈**（LIFO）：`push` 尾插，`pop` 尾删

注意：`table.remove(1)` 是 **O(n)** 操作，大队列用循环缓冲区。

### 代码示例（逐行注释）

```lua
-- 简单队列（FIFO）
local function create_queue()
    return {}
end

local function enqueue(q, item)
    table.insert(q, item)  -- 插入尾部
end

local function dequeue(q)
    return table.remove(q, 1)  -- 移除头部（O(n)！）
end

local function is_empty(q)
    return #q == 0
end

local q = create_queue()
enqueue(q, "task1")
enqueue(q, "task2")
enqueue(q, "task3")
print("dequeue:", dequeue(q))  -- task1
print("dequeue:", dequeue(q))  -- task2
print("is empty:", is_empty(q))

-- 双端队列（deque）
local function create_deque()
    return {}
end

local function push_front(q, item)
    table.insert(q, 1, item)  -- 头部插入
end

local function pop_back(q)
    return table.remove(q)  -- 尾部删除
end
```

## 5. table 库函数

### 概念讲解

table 库函数：

| 函数 | 说明 |
|------|------|
| `table.insert(t, pos, value)` | 插入 |
| `table.remove(t, pos)` | 删除 |
| `table.sort(t, comp)` | 排序 |
| `table.concat(t, sep, i, j)` | 连接 |
| `table.move(a, f, e, t)` | 移动 |
| `table.unpack` | 展开 |

### 代码示例（逐行注释）

```lua
local arr = {5, 2, 8, 1, 9, 3, 7, 4, 6}

-- sort
table.sort(arr)
print("sorted:", table.concat(arr, ", "))

-- sort with comparator
table.sort(arr, function(a, b) return a > b end)
print("sorted desc:", table.concat(arr, ", "))

-- concat
local words = {"hello", "world", "lua"}
print("concat:", table.concat(words, " "))
print("concat with indices:", table.concat(words, ", ", 2, 3))

-- unpack
local t = {1, 2, 3}
print("unpack:", table.unpack(t))

-- flatten（打平嵌套 table）
local nested = {{1, 2}, {3, 4}, {5, 6}}
local flat = {}
for _, sub in ipairs(nested) do
    for _, v in ipairs(sub) do
        table.insert(flat, v)
    end
end
print("flattened:", table.concat(flat, ", "))  -- 1, 2, 3, 4, 5, 6
```

## 6. 弱表（Weak Table）

### 生活类比

> 弱表就像"有自动清理功能的垃圾桶"——里面的东西如果没有别人引用，就会被垃圾车（GC）收走。没有弱表，缓存里的东西永远不会被清理。

### 概念讲解

弱表：引用可以被垃圾回收的 table。

设置：`setmetatable(t, {__mode = "k"})` 或 `"v"` 或 `"kv"`
- `"k"`：键是弱引用
- `"v"`：值是弱引用
- `"kv"`：键和值都是弱引用

用途：缓存、记忆化、避免循环引用

### 代码示例（逐行注释）

```lua
-- 弱引用值（value 是弱引用）
local cache = setmetatable({}, {__mode = "v"})

local key = {}
cache[key] = "expensive result"
print("cache[key]:", cache[key])

key = nil  -- 解除引用
collectgarbage()  -- 强制 GC

print("cache after GC:", next(cache))  -- nil（key 已被回收）

-- 记忆化示例
local function memoize(fn)
    local cache = setmetatable({}, {__mode = "kv"})
    return function(...)
        local key = table.concat({...}, ",", 1, select("#", ...))
        if cache[key] == nil then
            cache[key] = fn(...)
        end
        return cache[key]
    end
end

local function expensive(n)
    print("computing...")
    return n * 2
end

local memo = memoize(expensive)
print(memo(5))  -- computing... 10
print(memo(5))  -- 直接返回缓存
```
