## 1. Socket 基础

### 概念讲解

Lua 标准库没有内置 socket，需要安装 **luasocket** 库：
```bash
luarocks install luasocket
```

### 代码示例（概念）

```lua
-- 基本用法：
local socket = require("socket")

-- TCP 客户端：
local tcp = socket.connect("localhost", 80)
tcp:send("GET / HTTP/1.0\r\n\r\n")
local response = tcp:receive("*a")
tcp:close()

-- TCP 服务器：
local server = assert(socket.bind("*", 8080))
local client = server:accept()
local line = client:receive("*l")
client:send("Hello from server\n")
client:close()

-- UDP：
local udp = assert(socket.udp())
udp:setpeername("localhost", 8080)
udp:send("hello")
```

## 2. HTTP 请求

### 代码示例（概念）

```lua
local http = require("socket.http")
local ltn12 = require("socket.ltn12")

-- GET 请求
local response, status = http.request("http://example.com/")
print("status:", status)

-- POST 请求
local request_body = "key1=value1&key2=value2"
local response, status, headers = http.request{
    url = "http://example.com/post",
    method = "POST",
    headers = {
        ["Content-Type"] = "application/x-www-form-urlencoded",
        ["Content-Length"] = #request_body,
    },
    source = ltn12.source.string(request_body),
    sink = ltn12.sink.table(response_body),
}
```

## 3. 同步 vs 异步

### 概念讲解

- **luasocket**：同步（阻塞）IO
- **异步方案**：OpenResty（nginx + Lua）、LÖVE 游戏引擎、自定义事件循环 + 协程

---

## 本章小结

| 功能 | Lua 实现 |
|------|---------|
| TCP | luasocket（同步） |
| UDP | luasocket |
| HTTP | socket.http |
| 异步 | 协程 + 自定义事件循环 |
