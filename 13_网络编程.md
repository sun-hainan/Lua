## 1.1 生活类比：Socket 就像打电话

想象你要和朋友通电话，Socket 就是那部**电话机**。

| 电话场景 | Socket 对应 |
|---------|------------|
| 拿起电话拨号 | `socket.connect()` 建立连接 |
| 电话号码 = IP + 端口 | 服务器地址 + 端口号 |
| 对方接听 | 三次握手完成，连接建立 |
| 双方通话 | `send()` / `receive()` 数据传输 |
| 一方挂断 | `close()` 关闭连接 |
| 通话结束 | 资源释放 |

> **一句话理解**：Socket 是网络通信的"电话插口"，TCP 负责建立通话链路，UDP 则像发短信——不保证送达但更轻量。

## 1.2 TCP vs UDP：两种通信哲学

### TCP（传输控制协议）

```
可靠连接 → 三次握手建立 → 数据流传输 → 四次挥手断开
```

**特点：**
- 面向连接：通信前必须建立连接
- 可靠传输：丢包重传、数据校验、顺序保证
- 字节流：无固定边界，适合文件、网页
- 全双工：双方可同时读写

**类比：打电话**
- 拨号等对方接听（握手）
- 说话有回应，没听清要求重说
- 说完挂电话（挥手）

### UDP（用户数据报协议）

```
无连接 → 直接发送数据报 → 不保证到达
```

**特点：**
- 无连接：不需要握手，直接发
- 不可靠：不重传、不排序
- 数据报：有固定边界，每个数据包独立
- 轻量：Header 仅 8 字节

**类比：发短信**
- 直接发，不确认对方在线
- 每条短信独立，不保证顺序
- 发完不管，适合实时性场景

### TCP vs UDP 一图对比

```
         TCP                           UDP
    ┌──────────┐                ┌──────────┐
    │  面向连接 │                │  无连接   │
    │  可靠传输 │                │  不可靠   │
    │  字节流   │                │  数据报   │
    │  全双工   │                │  单向    │
    │  慢但稳   │                │  快但野  │
    └──────────┘                └──────────┘
```

## 1.3 Socket 原理：通信的底层逻辑

### 什么是 Socket？

Socket（套接字）是操作系统提供的**网络通信端点**，是应用程序与网络协议栈之间的接口。

```
应用程序  ←→  Socket API  ←→  TCP/UDP协议栈  ←→  网卡  ←→  互联网
   ↑            ↑               ↑                    ↑
  Lua代码     connect/        封包/解包           010101
  调用API     accept/send     拥塞控制            物理传输
```

### 端口号的作用

- **IP 地址**：定位某一台主机（类似家庭住址）
- **端口号**：定位主机上的某个应用程序（类似房间号）
- 范围：0 ~ 65535（其中 0~1023 为系统保留端口）

```
常见端口：
80   → HTTP 服务
443  → HTTPS 服务
3306 → MySQL 数据库
6379 → Redis 数据库
8080 → 常用开发测试端口
```

## 2.2 TCP 服务端：接听电话的服务器

### 概念讲解

TCP 服务端的工作流程：

```
1. 创建 socket 对象
2. 绑定地址和端口（bind）
3. 开始监听连接（listen）
4. 接受客户端连接（accept）
5. 与客户端通信（send / receive）
6. 关闭连接（close）
```

### 代码实现

```lua
-- ========== TCP 服务端 ==========
-- 第1步：加载 luasocket 库
local socket = require("socket")

-- 第2步：创建 TCP socket 对象
-- socket.tcp() 返回一个新的 TCP socket 实例
local server = assert(socket.tcp())

-- 第3步：绑定地址和端口
-- "*" 表示绑定所有网卡IP，"8080" 是监听端口
-- 第二个参数 backlog=50 表示最大排队连接数
server:setoption("reuseaddr", true)  -- 允许地址复用，避免端口占用
assert(server:bind("*", 8080))

-- 第4步：开始监听， accept 开始接受连接
server:listen(50)

print("TCP 服务器已启动，监听 0.0.0.0:8080")
print("等待客户端连接...")

-- 第5步：接受客户端连接（阻塞等待）
-- accept() 会一直阻塞，直到有客户端连接进来
-- 成功后返回一个 client socket 和客户端地址
local client, err = server:accept()
if not client then
    print("接受连接失败:", err)
    return
end

-- 获取客户端信息
local client_ip = client:getpeername()
print("客户端已连接:", client_ip)

-- 第6步：与客户端通信
-- 设置超时，避免永久阻塞
client:settimeout(10)  -- 10秒超时

-- 接收一行数据（遇到换行符为止）
local line, err = client:receive("*l")
if line then
    print("收到客户端消息:", line)

    -- 发送响应数据给客户端
    local response = "服务器收到: " .. line .. "\n"
    local sent, err = client:send(response)
    if not sent then
        print("发送失败:", err)
    end
end

-- 第7步：关闭连接（先关读，再关写）
client:shutdown("both")  -- 关闭读写双向
client:close()            -- 释放 socket 资源

-- 第8步：关闭服务器 socket
server:close()

print("服务器已关闭")
```

```

│              TCP 服务端运行流程               │
├─────────────────────────────────────────────┤
│                                             │

│         ↓                                   │
│  server:bind("*", 8080)    ←─ 绑定端口      │

│  server:listen(50)         ←─ 开始监听      │
```

## 2.3 TCP 客户端：拨打电话的客户端

### 代码实现

```lua
-- ========== TCP 客户端 ==========
-- 第1步：加载 luasocket 库
local socket = require("socket")

-- 第2步：创建 TCP socket 对象
local client = assert(socket.tcp())

-- 第3步：连接到服务器
-- 第一个参数：服务器 IP 或域名
-- 第二个参数：服务器端口
-- 连接失败会抛出错误，assert 用于捕获
print("正在连接服务器 localhost:8080 ...")
assert(client:connect("localhost", 8080))
print("连接成功！")

-- 第4步：设置超时
client:settimeout(10)

-- 第5步：发送数据给服务器
-- 注意：TCP 是字节流协议，数据可能分片发送
-- 通常自定义应用层协议（如换行符、长度前缀）
local message = "Hello, Server!\n"
local bytes_sent, err = client:send(message)
if not bytes_sent then
    print("发送失败:", err)
    return
end
print("已发送:", message:gsub("\n", ""))

-- 第6步：接收服务器响应
-- "*l" 表示读取一行（直到换行符）
-- "*a" 表示读取所有可用数据
local response, err = client:receive("*l")
if response then
    print("收到服务器响应:", response)
else
    print("接收失败或超时:", err)
end

-- 第7步：关闭连接
client:close()
print("连接已关闭")
```

### TCP 通信完整流程图

```
客户端                                    服务端
  │                                        │
  │ ──────── SYN (请求连接) ──────────→   │  第一次握手
  │                                        │
  │ ←────── SYN+ACK (同意连接) ───────    │  第二次握手
  │                                        │
  │ ─────── ACK (确认) ──────────────→   │  第三次握手
  │                                        │
  │ ═══════ 连接建立完成 ════════════════ │
  │                                        │
  │ ─────── send("Hello") ────────────→  │
  │                                        │
  │ ←────── receive("Hi") ─────────────   │
  │                                        │
  │ ═══════ 数据传输 ════════════════════ │
  │                                        │
  │ ─────── FIN (请求断开) ───────────→  │
  │                                        │
  │ ←────── ACK (确认断开) ────────────   │
  │ ←────── FIN (服务端断开) ──────────   │
  │ ─────── ACK (确认) ──────────────→   │
  │ ═══════ 连接关闭 ════════════════════ │
```

## 2.4 UDP 服务端：不敲门就送信

### 代码实现

```lua
-- ========== UDP 服务端 ==========
-- 第1步：加载 luasocket 库
local socket = require("socket")

-- 第2步：创建 UDP socket 对象
-- UDP 不需要 listen 和 accept
local server = assert(socket.udp())

-- 第3步：绑定地址和端口
-- UDPServerSetpeername 也可以用 setpeername 指定对端
server:setoption("reuseaddr", true)  -- 地址复用
assert(server:bind("*", 8080))

print("UDP 服务器已启动，监听 0.0.0.0:8080")

-- 第4步：设置超时（接收操作需要超时，否则永久阻塞）
server:settimeout(5)  -- 5秒超时

-- 第5步：循环接收数据报
-- UDP 的 receivefrom 返回 (数据, 对端IP, 端口)
for i = 1, 3 do  -- 最多接收3条消息
    print("等待接收数据...")
    local data, ip, port = server:receivefrom()
    if data then
        print(string.format("收到 [%s:%d]: %s", ip, port, data))

        -- 第6步：回发响应（UDP 可以回复给发送方）
        local reply = "服务端收到: " .. data
        server:sendto(reply, ip, port)
    else
        print("接收超时或出错")
        break
    end
end

-- 第7步：关闭 socket
server:close()
print("UDP 服务器已关闭")
```

## 2.5 UDP 客户端：发短信的人

### 代码实现

```lua
-- ========== UDP 客户端 ==========
-- 第1步：加载 luasocket 库
local socket = require("socket")

-- 第2步：创建 UDP socket 对象
local client = assert(socket.udp())

-- 第3步：设置默认对端地址
-- 这样后续可以直接用 send() 而不用 sendto()
assert(client:setpeername("localhost", 8080))

-- 第4步：设置超时
client:settimeout(3)

-- 第5步：发送数据报
local messages = {
    "你好，UDP 服务器！",
    "这是第二条消息",
    "再见！"
}

for _, msg in ipairs(messages) do
    print("发送:", msg)
    local bytes_sent, err = client:send(msg)
    if not bytes_sent then
        print("发送失败:", err)
        return
    end

    -- 第6步：接收服务器响应（可选，UDP 不保证有响应）
    -- 使用 settimeout(nil) 设为永久阻塞等待
    -- 这里注释掉，因为 UDP 服务器不一定回复
    -- local response, err = client:receive()
    -- if response then
    --     print("收到响应:", response)
    -- end

    -- 发送间隔
    socket.sleep(1)
end

-- 第7步：关闭 socket
client:close()
print("UDP 客户端已关闭")
```

## 3.1 HTTP 请求

### socket.http 用法

```lua
-- ========== HTTP GET 请求 ==========
local http = require("socket.http")
local ltn12 = require("socket.ltn12")

-- 简单 GET 请求
-- http.request() 返回 (响应体, 状态码, 响应头)
local response_body, status_code, headers = http.request("http://httpbin.org/get")
print("状态码:", status_code)
print("响应内容:", response_body)

-- ========== HTTP POST 请求 ==========
local post_body = "name=zhangsan&age=25"
local response_body = {}

local status_code, headers = http.request{
    url = "http://httpbin.org/post",
    method = "POST",
    headers = {
        ["Content-Type"] = "application/x-www-form-urlencoded",
        ["Content-Length"] = #post_body,
    },
    source = ltn12.source.string(post_body),
    sink = ltn12.sink.table(response_body),
}

print("POST 状态码:", status_code)
print("POST 响应:", table.concat(response_body))
```

## 3.2 同步 vs 异步

### 同步模型（luasocket 默认）

```lua
-- 同步调用：send() 会阻塞，直到数据发送完成
-- 适合：命令行工具、脚本、简单服务器
local client = assert(socket.tcp())
client:connect("localhost", 8080)
client:send("hello")  -- 阻塞，直到数据发出或出错
local response = client:receive()  -- 阻塞，直到收到数据
```

### 异步模型：协程 + 事件循环

```lua
-- 异步思路：用协程模拟非阻塞，让出执行权
-- 适合：高并发服务器、游戏服务器

-- 伪代码示例
local connections = {}
local socket = require("socket")

local server = assert(socket.tcp())
server:bind("*", 8080)
server:listen(100)
server:settimeout(0)  -- 非阻塞模式

while true do
    -- 检查是否有新连接
    local client = server:accept()
    if client then
        client:settimeout(0)
        table.insert(connections, {socket = client, last_active = os.time()})
    end

    -- 处理所有连接的数据
    for i = #connections, 1, -1 do
        local conn = connections[i]
        local data, err = conn.socket:receive()

        if data then
            -- 处理数据
            process_data(conn, data)
        elseif err == "timeout" then
            -- 忽略超时
        else
            -- 连接断开
            conn.socket:close()
            table.remove(connections, i)
        end
    end

    socket.sleep(0.01)  -- 让出 CPU
end
```

## 4.1 常见错误

| 错误 | 原因 | 解决方法 |
|-----|------|---------|
| `Timeout error` | `receive()` 超时 | 设置合理的 `settimeout()` |
| `Connection refused` | 服务端未启动或端口不对 | 检查服务器是否在运行 |
| `Address already in use` | 端口被占用 | `setoption("reuseaddr", true)` |
| 数据粘包 | TCP 字节流无边界 | 应用层加长度前缀或分隔符 |
| 死锁 | 双向读写没有超时 | 每条读写都要设超时 |
| 端口忘记关闭 | 资源泄漏 | 确保 `close()` 在 `pcall` 或 `finally` 中调用 |
