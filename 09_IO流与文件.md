## 1. 文件读写

### 生活类比

> 文件操作就像"去图书馆借书还书"——`io.open` 是"找管理员开门"，`file:read` 是"翻书页"，`file:write` 是"在书页上写字"，`file:close` 是"把书还回去"。

### 概念讲解

`io.open(filename, mode)` 打开文件：

| 模式 | 说明 |
|------|------|
| `"r"` | 读（默认） |
| `"w"` | 写（覆盖） |
| `"a"` | 追加 |
| `"r+"` | 读写 |
| `"b"` | 二进制模式 |

### 代码示例（逐行注释）

```lua
-- 写文件
local file = io.open("test_output.txt", "w")
if file then
    file:write("Hello, Lua!\n")
    file:write("Second line\n")
    file:close()
    print("wrote to test_output.txt")
end

-- 读文件
file = io.open("test_output.txt", "r")
if file then
    local content = file:read("*a")  -- 读全部
    print("file content:")
    print(content)
    file:close()
end

-- 逐行读取
file = io.open("test_output.txt", "r")
if file then
    for line in file:lines() do
        print("line:", line)
    end
    file:close()
end
```

### 常见错误

```lua
-- ❌ 文件不存在时 read 不检查
-- file = io.open("nonexistent.txt", "r")
-- local content = file:read("*a")  -- 可能崩溃

-- ✅ 先检查
local file, err = io.open("nonexistent.txt", "r")
if not file then
    print("error:", err)
end
```

## 2. 标准输入输出

### 概念讲解

- `print()`：自动加换行
- `io.write()`：不加换行
- `io.read()`：读取标准输入
- `io.input()` / `io.output()`：设置默认文件

### 代码示例（逐行注释）

```lua
-- print 自动加换行
print("Hello from print")

-- io.write 不加换行
io.write("Hello from io.write")
io.write(" - no newline\n")

-- 读取模式
-- io.read("*l")    读一行
-- io.read("*a")    读全部
-- io.read("*n")    读数字
-- io.read()        同 "*l"
```

## 3. 二进制文件处理

### 概念讲解

二进制模式：打开时加 `"b"`

### 代码示例（逐行注释）

```lua
-- 写入二进制数据
local file = io.open("binary.bin", "wb")
if file then
    local bytes = {0x48, 0x65, 0x6C, 0x6C, 0x6F}  -- "Hello"
    for _, b in ipairs(bytes) do
        file:write(string.char(b))
    end
    file:close()
    print("wrote binary file")
end

-- 读取二进制数据
file = io.open("binary.bin", "rb")
if file then
    local bytes = {}
    while true do
        local byte = file:read(1)
        if not byte then break end
        table.insert(bytes, string.byte(byte))
    end
    file:close()
    print("read bytes:", table.concat(bytes, ", "))
end
```

## 4. 序列化与反序列化

### 生活类比

> 序列化就像"把家具拆了装进箱子"——Lua table 变成字符串，可以存文件、发网络。反序列化就是"打开箱子把家具装回去"。

### 代码示例（逐行注释）

```lua
-- 简单序列化
local function serialize(o)
    local t = type(o)
    if t == "nil" then return "nil"
    elseif t == "number" then return tostring(o)
    elseif t == "string" then return string.format("%q", o)
    elseif t == "boolean" then return tostring(o)
    elseif t == "table" then
        local parts = {}
        for k, v in pairs(o) do
            table.insert(parts, string.format("[%s] = %s",
                serialize(k), serialize(v)))
        end
        return "{" .. table.concat(parts, ", ") .. "}"
    else
        return tostring(o)
    end
end

local data = {
    name = "Alice",
    age = 30,
    scores = {95, 87, 92},
    active = true,
}

local s = serialize(data)
print("serialized:")
print(s)

-- 反序列化（loadstring）
local loaded = load("return " .. s)()
print("deserialized:", loaded.name, loaded.age, loaded.scores[1])
```

## 本章小结

| 功能 | Lua 实现 |
|------|---------|
| 打开文件 | `io.open(filename, mode)` |
| 读 | `file:read("*a")` / `file:lines()` |
| 写 | `file:write(str)` |
| 关闭 | `file:close()` |
| 刷新 | `file:flush()` |
| 二进制模式 | `"rb"` / `"wb"` |
| 序列化 | 手写或库（penlight 等） |
