## 1. 变量声明：全局 vs 局部

### 生活类比

> 全局变量就像贴在冰箱上的便利贴——所有人都能看到，所有人都能改，很容易被乱贴乱撕。局部变量就像每个人口袋里的私人物品——只有你自己能用，不会被别人打扰。

### 概念讲解

Lua 变量：
- **全局变量**：直接赋值，无需声明（挂载在 `_G` 表中）
- **局部变量**：`local` 关键字声明（推荐）

**推荐始终使用 `local`**，原因：
- local 访问更快（寄存器 vs 全局表查找）
- 避免命名冲突
- 作用域明确，代码更易维护

### 代码示例（逐行注释）

```lua
-- 局部变量（推荐）
local name = "Alice"
print("local name:", name)          -- Alice

-- 全局变量（不推荐）
age = 30                            -- 全局变量，挂在 _G 表中
print("global age:", age)           -- 30

-- 多个变量赋值
local x, y, z = 1, 2, 3
print("multiple:", x, y, z)         -- 1  2  3

-- 全局变量实际上是 _G 表的键
print("_G.age == age:", _G.age == age)  -- true
```

### ASCII 图示

```
全局变量 vs 局部变量：

全局作用域 (_G):
┌─────────────────────────────────────┐
│  age = 30          ──┐              │
│  name = "global"    ──┤  所有函数   │
│                      ──┤  都能访问   │
└─────────────────────────────────────┘

局部作用域：
function outer()
    local outer_local = "outer"    ◄─── 只有 outer 内部能访问
    function inner()
        local inner_local = "inner"  ◄── 只有 inner 能访问
        print(outer_local)           ◄── inner 可以访问 outer 的局部
    end
end
```

### 常见错误

```lua
-- ❌ 全局变量命名冲突
function foo()
    result = 42  -- 不小心创建了全局变量
end

-- ✅ 正确
function foo()
    local result = 42
end
```

## 2. Lua 的 8 种基本类型

### 生活类比

> Lua 的类型就像厨房里的8种食材——有蔬菜（nil）、水果（boolean）、主食（number）、调料（string）等。每种"食材"有不同的用法。

### 概念讲解

Lua 8 种基本类型：

| 类型 | 说明 | 示例 |
|------|------|------|
| `nil` | 空值 | `nil` |
| `boolean` | 布尔 | `true`, `false` |
| `number` | 双精度浮点 | `42`, `3.14`, `0xFF` |
| `string` | 不可变字节序列 | `"hello"`, `'world'` |
| `function` | 函数（一等公民） | `function() end` |
| `table` | 唯一复合类型 | `{}` |
| `userdata` | C 结构 | 无法直接创建 |
| `thread` | 协程 | `coroutine.create()` |

### 代码示例（逐行注释）

```lua
-- type() 查看类型
print(type(nil))              -- nil
print(type(true))            -- boolean
print(type(42))              -- number
print(type("hello"))         -- string
print(type(function() end))  -- function
print(type({}))              -- table
print(type(io.stdout))       -- userdata
print(type(coroutine.create(function() end)))  -- thread
```

### 常见错误

```lua
-- ❌ 混淆字符串和数字
print(1 == "1")              -- false（类型不同）

-- ✅ tonumber 显式转换
print(tonumber("1") == 1)    -- true
```

## 3. Table：唯一的复合类型

### 生活类比

> table 就像一个万能收纳盒——可以装苹果（数组模式）、装文件（字典模式）、甚至装盒子里的盒子（嵌套）。它是 Lua 唯一的数据结构，却能做所有事情。

### 概念讲解

table 是 Lua **唯一的复合类型**，可以：
- **数组**（索引从 1 开始）
- **字典/映射**
- **对象/结构体**
- **命名空间**

核心特性：键值对存储、动态大小、引用语义

### 代码示例（逐行注释）

```lua
-- 数组用法（索引从 1 开始，不是 0！）
local arr = {10, 20, 30, 40, 50}
print("arr[1]:", arr[1])          -- 10（不是 arr[0]！）
print("arr length:", #arr)       -- 5

-- 字典用法
local dict = {name = "Alice", age = 30}
print("dict.name:", dict.name)              -- Alice
print("dict['age']:", dict["age"])          -- 30

-- 混合用法（同一个 table 可以同时有数组部分和字典部分）
local mixed = {
    "string",                     -- arr[1]
    42,                          -- arr[2]
    key = "value",               -- dict.key
    nested = {a = 1, b = 2}      -- dict.nested（嵌套 table）
}
print("mixed[1]:", mixed[1])      -- string
print("mixed.key:", mixed.key)    -- value
print("mixed.nested.a:", mixed.nested.a)  -- 1
```

### ASCII 图示

```
table 的三种角色：

1. 数组（Array）：
   arr = {10, 20, 30}
   ┌────┬────┬────┐
   │ 1  │ 2  │ 3  │  ← 索引（从 1 开始！）
   ├────┼────┼────┤
   │ 10 │ 20 │ 30 │  ← 值
   └────┴────┴────┘

2. 字典（Dictionary）：
   dict = {name="Alice", age=30}
   ┌────────┬────────┐
   │ "name" │ "Alice"│
   ├────────┼────────┤
   │ "age"  │  30    │
   └────────┴────────┘

3. 对象（Object）：
   obj = setmetatable({x=1, y=2}, {__index = methods})
```

### 常见错误

```lua
-- ❌ 数组索引从 0 开始（初学者常犯）
local arr = {1, 2, 3}
print(arr[0])                    -- nil！

-- ✅ 正确：从 1 开始
print(arr[1])                    -- 1

-- ❌ 混淆 nil 和 false
local t = {flag = false}
print(t.flag == nil)            -- false（键存在，值是 false）
print(t.missing == nil)         -- true（键不存在）
```

## 4. 作用域规则与 upvalue

### 生活类比

> upvalue 就像"时光胶囊"——内部函数可以访问创建时打包进去的外部变量。即使外部函数已经执行完毕，那个变量依然被内部函数"记住"。

### 概念讲解

Lua 作用域层级：
1. **全局作用域**（`_G`）
2. **模块作用域**（文件级 local）
3. **函数作用域**
4. **块作用域**（if/for/while 内）

**upvalue**：内部函数引用外部局部变量

### 代码示例（逐行注释）

```lua
local module_level = "module"

local function outer()
    local outer_local = "outer"

    local function inner()
        local inner_local = "inner"
        -- upvalue：inner 可以访问 outer_local
        print("inner sees outer_local:", outer_local)   -- outer
        print("inner sees module_level:", module_level) -- module
    end

    inner()
    -- print(inner_local)  -- ❌ outer 无法访问 inner 的局部变量
end

outer()

-- 块级作用域示例
if true then
    local block_var = "in block"
    print("inside block:", block_var)
end
-- print(block_var)  -- ❌ 块外无法访问
```

### ASCII 图示

```
作用域层级（同心圆，从外到内）：

┌─────────────────────────────────────────┐  ← 全局 _G
│  module_level = "module"               │
│  ┌───────────────────────────────────┐  │
│  │  outer_local = "outer"           │  │  ← 函数作用域
│  │  ┌─────────────────────────────┐ │  │
│  │  │  inner_local = "inner"      │ │  │  ← 更内层函数
│  │  │  print(outer_local) ✓       │ │  │  ← upvalue 访问
│  │  │  print(inner_local) ✓        │ │  │
│  │  └─────────────────────────────┘ │  │
│  │  print(inner_local) ✗            │ │  ← outer 不能访问 inner
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

### 常见错误

```lua
-- ❌ 误以为闭包中的变量是共享的（实际上是各自的副本）
local function make_counter()
    local count = 0
    return function()
        count = count + 1
        return count
    end
end

local c1 = make_counter()
local c2 = make_counter()
print(c1())  -- 1（c1 的独立 count）
print(c1())  -- 2
print(c2())  -- 1（c2 有自己的 count）
```

## 5. nil 与 false 的区别

### 生活类比

> nil 像是"没有这个抽屉"，false 像是"抽屉在，但里面装着『假』"。不存在和存在但是假的，是两回事。

### 概念讲解

都是"假"值，但语义不同：
- **nil**：表示"没有值"，未定义
- **false**：明确是"假"

使用场景：
- nil：可选参数默认值、未找到的结果
- false：明确的布尔状态

### 代码示例（逐行注释）

```lua
local not_set = nil               -- 未定义
local explicitly_false = false     -- 明确为假

-- 都是 falsy
print("nil is falsy:", not not_set)              -- true
print("false is falsy:", not explicitly_false)   -- true

-- 但 == 比较永远不等
print("nil == false:", nil == false)             -- false

-- nil 用来"删除"
local temp = 100
temp = nil  -- 删除变量

-- false 用来"设置状态"
local is_enabled = false
print("is_enabled:", is_enabled)
```

### 常见错误

```lua
-- ❌ 混淆
if value == false then
    -- 这永远不会命中 nil 的情况
end

-- ✅ 正确判断
if value == nil then
    -- 只有 nil 才会进来
elseif value == false then
    -- 只有 false 才会进来
end
```

## 6. 进制与数字类型

### 生活类比

> 数字 42 可以写成"42"（十进制）、"0x2A"（十六进制）、"0b101010"（二进制）——只是表示方式不同，本质都是同一个数。

### 概念讲解

Lua number 是**双精度浮点数**（IEEE 754 64位），支持多种进制字面量：
- 十进制：`42`
- 十六进制：`0xFF`（255）
- 二进制：`0b1010`（Lua 5.3+）
- 八进制：`0o77`（Lua 5.3+）
- 科学计数法：`1.5e10`

### 代码示例（逐行注释）

```lua
-- 多种进制
local decimal = 42
local hex = 0xFF                  -- 255
local binary = 0b1010             -- 10 (Lua 5.3+)
local octal = 0o77                -- 63 (Lua 5.3+)
local scientific = 1.5e10

-- 进制转换
local str_num = "255"
local converted = tonumber(str_num)
print("tonumber('255'):", converted)           -- 255

-- 带进制转换
print("tonumber('FF', 16):", tonumber("FF", 16))  -- 255
print("tonumber('1010', 2):", tonumber("1010", 2))  -- 10

-- 数字转字符串
local formatted = string.format("%.2f", 3.14159)
print("formatted:", formatted)               -- 3.14
```

### 常见错误

```lua
-- ❌ Lua 5.2 及以下不支持二进制和八进制字面量
-- print(0b1010)  -- 语法错误（Lua 5.2）

-- ❌ 进制转换第二个参数写错
print(tonumber("FF", 10))  -- nil（十进制没有 F）
print(tonumber("FF", 16))  -- 255
```

## 7. 字符串操作与类型转换

### 生活类比

> Lua 字符串是不可变的"刻字石头"——你不能修改一块石头上的字，只能刻一块新石头。`..` 拼接就是"把两块石头粘在一起变成新石头"。

### 概念讲解

- **字符串不可变**：每次"修改"实际是创建新字符串
- **拼接**：`..`（两个点）
- **长度**：`#str`
- **切片**：`string.sub(s, i, j)`
- **查找**：`string.find(s, pattern)`
- **替换**：`string.gsub(s, pattern, replacement)`

Lua 自动转换：
- 数字和字符串在运算时自动转换
- `tostring()` 数字转字符串
- `tonumber()` 字符串转数字

### 代码示例（逐行注释）

```lua
local s = "Hello, World!"

-- 长度
print("length:", #s)                       -- 13

-- 拼接
local greeting = "Hello" .. ", " .. "World!"
print("concatenation:", greeting)

-- 切片
print("sub(1,5):", s:sub(1, 5))           -- Hello
print("sub(-6):", s:sub(-6))              -- World!

-- 大小写
print("upper:", s:upper())
print("lower:", s:lower())

-- 查找
local start, ends = s:find("World")
print("find 'World':", start, ends)       -- 8  12

-- 替换
print("gsub:", s:gsub("World", "Lua"))    -- Hello, Lua!  1

-- 字符串拼接自动转换数字
local num = 42
print("string concat:", num .. " is the answer")  -- "42 is the answer"

-- 算术运算自动转换字符串数字
print("string + number:", "123" + 1)       -- 124

-- 显式转换
local function explicit_conversion()
    local num = 42
    local str = tostring(num)
    print("tostring(42):", str, type(str)) -- 42  string

    local back = tonumber(str)
    print("tonumber('42'):", back, type(back))  -- 42  number

    local invalid = tonumber("hello")
    print("tonumber('hello'):", invalid)    -- nil
end

explicit_conversion()
```

### ASCII 图示

```
字符串切片：
s = "Hello, World!"
位置: 1 2 3 4 5 6 7 8 9 ...
     H e l l o ,   W o r l d !

s:sub(1, 5)  → "Hello"
s:sub(-6)    → "World!"
s:sub(8, -1) → "World!"
```

### 常见错误

```lua
-- ❌ 字符串拼接不是算术运算
-- print("hello" + 1)  -- ❌ 错误：尝试对字符串做算术

-- ❌ 忘记 tonumber 可能返回 nil
local n = tonumber("hello")
-- if n > 0 then  -- ❌ 可能报错
-- ✅ 应该检查
if n then
    print(n + 1)
else
    print("invalid number")
end
```

## 本章小结

| 类型 | Lua 说明 |
|------|---------|
| `nil` | 空值，删除变量 |
| `boolean` | `true` / `false`（注意：nil 也是 falsy） |
| `number` | 双精度浮点数，支持多进制 |
| `string` | 不可变字节序列，用 `..` 拼接 |
| `table` | 唯一复合类型，可作数组/字典/对象 |
| `function` | 一等公民，可存储/传递/闭包 |
| `userdata` | C 结构体 |
| `thread` | 协程 |
| 字符串长度 | `#s` 或 `string.len(s)` |
| 数组长度 | `#t`（索引从 1 开始！） |
| 类型转换 | `tostring()` / `tonumber()` |
